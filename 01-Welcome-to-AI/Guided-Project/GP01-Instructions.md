## Common Issues & Solutions

**Problem:** "NameError: name 'temperatures' is not defined"  
**Solution:** Run the cell that creates the `temperatures` list first

**Problem:** Notebook won't save  
**Solution:** Make sure you saved a copy to your Google Drive first

**Problem:** Output looks weird  
**Solution:** Click **Runtime ‚Üí Restart and run all** to start fresh

**Problem:** Code has errors  
**Solution:** Check for typos - Python is case-sensitive and spacing matters!
